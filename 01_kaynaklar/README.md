# 01 — Kaynaklar

Bu dizin kaynak belgelerin hedef sınıflandırma alanıdır. Mevcut ham belgeler henüz taşınmamış, silinmemiş veya yeniden adlandırılmamıştır.

## Alt dizinler

- `yonetmelikler/`
- `prosedurler/`
- `yonergeler/`
- `gorev_tanimlari/`
- `formlar/`
- `is_akislari/`
- `mevzuat/`

Kaynak taşıma işlemi yapılırken orijinal dosya adı, doküman kodu, revizyon ve kaynak yolu envanterde korunmalıdır.
